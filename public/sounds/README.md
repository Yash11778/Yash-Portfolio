# Sound Effects

This directory contains sound effects for interactive elements in the portfolio.

## Required Sound Files

You need to add the following MP3 files to this directory:

1. **click.mp3** - Button click sound
2. **hover.mp3** - Hover interaction sound
3. **success.mp3** - Success notification sound (form submission, etc.)
4. **achievement.mp3** - Achievement unlock sound
5. **type.mp3** - Typing sound effect
6. **whoosh.mp3** - Page transition or scroll sound
7. **pop.mp3** - Pop/bubble sound for UI interactions

## Free Sound Resources

You can download free sound effects from:

- **Pixabay Sounds**: https://pixabay.com/sound-effects/
- **Freesound**: https://freesound.org/
- **Zapsplat**: https://www.zapsplat.com/
- **Mixkit**: https://mixkit.co/free-sound-effects/

## Recommended Settings

- Format: MP3
- Sample Rate: 44.1 kHz
- Bit Rate: 128 kbps
- Duration: 0.2 - 1.0 seconds (short and subtle)
- Volume: Normalized to avoid clipping

## Usage

The sound effects are automatically loaded and played by the `useSoundEffects` hook.
Users can toggle sound effects on/off via the settings (feature to be implemented).
